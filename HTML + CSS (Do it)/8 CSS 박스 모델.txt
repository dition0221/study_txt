8-1 CSS와 박스 모델

블록 레벨 요소
= 요소를 삽입했을 때 혼자 한 줄을 차지하는 요소
- 요소의 너비가 100%
ex ) <div>, <p> 등

인라인 레벨 요소
= 줄을 차지하지 않는 요소
- 화면에 표시되는 콘텐츠만큼만 영역을 차지하고, 나머지 공간에는 다른 요소가 올 수 있음
ex ) <img>, <strong>, <span> 등

박스 모델
- 실제 콘텐츠 영역, 패딩(padding), 박스의 테두리(border), 마진(margin) 등의 요소로 구성됨

width, height 속성
= 실제 콘텐츠 영역의 크기 지정
= width : <크기> | <백분율> | auto
= height : <크기> | <백분율> | auto
	- <크기> : px 또는 cm 단위
	- <백분율> : 부모 요소를 기준으로 백분율(%)로 지정
	- auto : [기본 값] 콘텐츠 양에 따라 자동으로 결정

실제 콘텐츠 크기 계산
- 모던 브라우저에서 박스 모델의 전체 너비
	= width 값 + 좌우 패딩 + 좌우 테두리
- 인터넷 익스플로러 6에서 박스 모델의 width 값
	= 콘텐츠 너비 + 좌우 패딩 + 좌우 테두리

display 속성
= 블록 레벨 요소와 인라인 레벨 요소를 서로 바꿈
- 원래는 해당 요소가 화면에 어떻게 보일지를 지정할 때 사용
1. block 속성 값
= 해당 요소를 블록 레벨로 지정
2. inline 속성 값
= 해당 요소를 인라인 레벨로 지정
- 너비나 높이, 위 아래 마진, float 같은 값이 정확히 적용되지 않음
3. inline-block 속성 값
= 해당 요소를 인라인 레벨로 배치하면서, 내용에는 블록 레벨 속성을 지정
- 블록 레벨 요소와 인라인 레벨 요소 두 가지 특성을 모두 가짐
4. none 속성 값
= 해당 요소를 화면에 표시하지 않음
- 화면에서 공간도 차지하지 않음
- 반응형 웹 디자인 기법으로 작성할 때 PC에서는 표시하지만, 모바일에서는 보이지 않도록 할 때 사용



8-2 테두리 관련 속성들

border-style 속성
= 테두리(border) 스타일을 지정
	- none : [기본 값] 화면에 테두리가 표시 안 됨
	- solid : 실선
	- dotted : 점선
	- dashed : 짧은 선(직선으로 된 점선)
- border-<방향>-style 로도 사용 가능 { top, bottom, right, left }

border-width 속성
= 테두리의 두께 지정
= border-width : <크기> | thin | medium | thick
- border-<방향>-width 로도 사용 가능 { top, bottom, right, left }
- 한꺼번에 지정 가능
	- 1개 : 네 방향에 모두 적용
	- 2개 : 위아래, 좌우 묶어서 적용
	- 3개 : 위, 좌우, 아래 각각 적용
	- 4개 : 위부터 시계방향으로 적용
		ex ) .box3 { border-width : 5px 10px 15px 20px; }

border-color 속성
= 테두리의 색상 지정
- border-<방향>-color 로도 사용 가능 { top, bottom, right, left }

border 속성
= 테두리 스타일들을 묶어서 지정
- border-<방향> 로도 사용 가능 { top, bottom, right, left }
- 순서는 상관없음

border-radius 속성
= 박스 모서리 부분을 둥글게 처리
= border-radius : <크기> | <백분율>
	- 원의 반지름(r; radius) 값을 지정
- border-<방향(top, bottom)>-<방향(left, right)>-radius 로도 사용 가능
- 가로 반지름과 세로 반지름의 크기를 함께 지정해, 타원형 모양을 만들 수 있음
	- 각 모서리마다 따로 지정한다면, 차례로 입력
	- 네 방향을 한꺼번에 지정한다면, 슬래시(/)를 넣어 구분

box-shadow 속성
= 선택한 요소에 그림자 효과 내기
= box-shadow : none | <그림자 값> [, <그림자 값>]*;
   <그림자 값> = <수평 거리> <수직 거리> <흐림정도> <번짐 정도> <색상> inset
	- <수평 거리>, <수직 거리 > : [필수]



8-3 여백을 조절하는 속성들

padding 속성
= 콘텐츠 영역과 테두리 사이의 여백 (테두리 안쪽 여백)
= padding : <크기> | <백분율> | auto
- padding-<방향> 로도 사용 가능 { top, bottom, right, left }
- 한꺼번에 지정 가능

margin 속성
= 현재 요소 주변의 여백
- 요소와 요소 간의 간격 조절 가능
= margin : <크기> | <백분율> | auto
- margin-<방향> 로도 사용 가능 { top, bottom, right, left }
- 한꺼번에 지정 가능
- 좌우 margin을 auto로 지정하면, 요소의 너비 값을 뺀 나머지 공간의 좌우 margin을 똑같이 맞춤
	- 웹 요소를 중앙에 배치하려고 할 때 자주 사용

마진 중첩(margin overlap) 현상 (= 마진 상쇄(margin collapse))
= 요소를 세로로 배치할 경우, 마진과 마진이 만날 때 마진 값이 큰 쪽으로 겹쳐지는 현상
- 여러 요소들을 세로로 배치할 때, 중간에 있는 마진들이 너무 커지는 것을 방지하기 위함
- 오른쪽 마진과 왼쪽 마진이 만날 경우에는 중첩되지 않음
