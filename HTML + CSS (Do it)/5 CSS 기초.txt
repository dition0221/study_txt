5-1 스타일과 스타일 시트

스타일(style)
= 미리 약속한 스타일 속성을 입력해 웹을 디자인하는 것

스타일 시트
= 여러 개의 스타일들을 모아 놓은 것
- 내부 스타일 시트
	- 웹 문서를 화면에 표시하기 전에 결정되어야 함
	- <head>태그 안에서 정의
	- <style>태그 사이에 작성
- 외부 스타일 시트 : 별도 파일에 저장 후 링크로 연결해 사용
	- 확장자 .css
	- <style>태그 없이, <link>태그만 사용해 외부 스타일 시트 파일을 연결
	= <link rel="stylesheet" href="외부 스타일 파일 경로">
- 인라인 스타일
	- 해당 태그에만 스타일 지정
	- style="속성 : 속성값;" 형태로 속성을 사용

스타일 형식
= 선택자 { 스타일 속성 : 속성 값; }



5-2 주요 선택자

전체 선택자 (Universal Selector)
= 스타일을 모든 요소에 적용할 때 사용
= *(별표)를 사용
- 주로 모든 하위 요소에 한꺼번에 스타일을 적용할 때 사용
- 문서의 여백이나 글꼴 크기 등 기본 스타일을 초기화할 때 사용
	- 마진(문서 바깥 여백)이나 패딩(문서 안쪽 여백) 등

태그 선택자
= 특정 태그를 사용한 모든 요소에 스타일을 적용

클래스 선택자
= 마침표(.) 다음에 클래스 이름 지정
	ex ) .bluetext { color : blue; }
- class 속성을 사용
- 다른 선택자를 선택해서 클래스의 범위를 제한이 가능
- 여러 번 재사용이 가능
- <span></span>태그를 사용해, 텍스트 일부에만 클래스 스타일을 적용 가능

id 선택자
= 샵(#) 다음에 id명을 지정
= #아이디명 { 스타일 }
- id 속성을 이용
- 문서 안에서 한 번만 사용이 가능

그룹 선택자
= 둘 이상의 선택자를 함께 지정
	ex ) h1, h2 { 스타일 속성 : 속성 값; }
- 쉼표(,)로 구분해 여러 선택자를 나열



5-3 캐스케이딩 스타일 시트(CSS)

캐스캐이딩
= 폭포처럼 '위에서 아래로 흐른다'는 뜻
- 선택자에 여러 스타일이 적용될 때, 스타일 충돌을 막기위한 방법
- 충돌을 막기위한 원칙
	- 스타일 상속
	- 스타일 우선순위

스타일 상속
- 자식 요소에서 별도로 스타일을 지정하지 않으면, 부모 요소에 있는 스타일 속성들이 자식 요소로 전달됨
- 상속을 이용하면 스타일 시트를 효과적으로 만들 수 있음
** 스타일의 모든 속성이 부모 요소에서 자식 요소로 상속되는 것은 아님
	ex ) 글자색은 상속되지만, 배경색과 이미지는 상속되지 않음 (기본 값인 '투명'으로 지정)
- 부모 요소로부터 스타일이 상속되는데 자식 요소에서 다른 스타일을 사용하고자 한다면, 즉 스타일 충돌이 생긴다면 '중요도', '명시도', '소스순서' 등에 따라 우선순위가 결정

스타일 우선순위
- 얼마나 중요한가에 따라
	1. 사용자 스타일 시트의 중요 스타일 - 시스템에서 만든 스타일 / 사용자가 제어 불가
	2. 제작자 스타일 시트의 중요 스타일 - !important를 붙임
	3. 제작자 스타일 시트의 일반 스타일
	4. 브라우저 스타일 시트의 스타일 - 브라우저의 기본 스타일
- 얼마나 한정지을 수 있는가에 따라 (적용되는 범위에 따라)
	1. 인라인 스타일
	2. id 스타일
	3. 클래스 스타일
	4. 태그 스타일
- 소스 순서에 따라
	- 소스에서 나중에 온 스타일이 먼저 온 스타일을 덮어씀



5-4 CSS3와 CSS 모듈

CSS 모듈
= CSS3부터 기능들을 주제별로 따로 만든 규약들
- CSS2 규약에는 한 번에 담겨져 있었음
- 각각의 모듈은 업데이트가 따로 진행됨
- CSS3는 한 번에 표준 규약이 결정되지 않음

CSS3와 브라우저 접두사(prefix)
- 표준 규약이 아닌 또는 이전 버전의 브라우저 사용자를 위한 속성들은 브라우저에 따라 다른 방식으로 지원되기 때문에, 속성 이름 앞에 접두사(prefix)를 붙여 브라우저별로 구분해야 함
- 브라우저 접두사 자동으로 붙이기
	- '-prefix-free'라는 자바스크립트 파일을 이용
	- http://leaverou.github.io/prefixfree/ 사이트에 접속한 후 [Only 2KB gzipped]를 클릭해 js파일을 다운로드 받기
	- html파일에서 <script>태그를 이용해 prefixfree.min.js 파일을 연결
		<script src="prefixfree.min.js"></script>
