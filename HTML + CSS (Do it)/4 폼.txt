4-1 폼 만들기

폼(form)
= 사용자가 웹 사이트로 정보를 보낼 수 있는 요소들

<form> 태그
= 폼을 만드는 가장 기본적인 태그
- 속성
	- method	: 사용자가 입력한 내용들을 서버 쪽 프로그램으로 어떻게 넘겨줄지 지정
		- get : URL에 사용자가 입력한 내용이 그대로 보여짐
		- post : 대부분 이 방식을 사용 / 암호화
	- name : 폼의 이름
	- action : <form> 태그 안에 내용들을 처리해 줄 서버 상의 프로그램을 지정
	- autocomplete : 자동완성기능 / 기본값 "on"

<label> 태그
= 폼 요소에 레이블(텍스트)을 붙이는 태그
- for 속성 : 동기화

<fieldset> 태그
= 하나의 폼 안에서 여러 구역을 나누어 표시할 때
- 폼들을 하나의 영역으로 묶고, 외곽선을 그려줌

<legend> 태그
= <fieldset> 태그로 묶은 그룹에 제목을 붙여줌



4-2 사용자 입력을 위한 <input> 태그

<input> 태그
= 사용자가 내용을 입력하는 필드를 삽입할 때나 버튼을 삽입할 때 사용
- type 속성 { text, submit, radio 등 }



4-3 <input> 태그의 다양한 속성

autofocus 속성
= 페이지를 불러오자마자 폼의 요소 중에서 원하는 요소에 마우스 커서를 표시
- [불린]

placeholder 속성
= 힌트 표시

readonly 속성
= 읽기 전용
- [불린]

required
= 필수 필드로 지정
- [불린]



4-4 여러 데이터 나열해 보여 주기

<select> 태그
= 드롭다운 목록 생성
	- size : 화면에 표시될 드롭다운 메뉴 개수를 지정
	- multiple : [불린] 다수 선택 가능

<option> 태그
= 드롭다운 목록의 옵션을 하나하나 생성
	- value : 서버로 넘겨질 값
	- selected : [불린] 처음 선택되어 있는 값

<optgroup> 태그
= 옵션의 항목들을 그룹으로 묶을 때 사용
	- label : 그룹 제목 붙임

<textarea> 태그
= 여러 줄 입력하는 텍스트 영역
	- cols : 가로 너비
	- rows : 세로 너비 (줄 단위)



4-5 기타 다양한 폼 요소들

<button> 태그
= 버튼 삽입
	- type 속성 { submit, reset, button }
	- button : 기능 X / JS로 기능 추가
- CSS로 style 변경 가능
- 버튼을 이미지 파일로 만들어 <input type="image">로 넣었을 때 보다 빠름

<output> 태그
- 화면에 다르게 표시되진 않지만, 웹 브라우저가 결과값이라는 것을 인식함

<profress> 태그
= 작업 진행 상태 표시

<meter> 태그
= 지정된 범위 내에서 해당 값이 어느 정도 차지하고 있는지를 표현
